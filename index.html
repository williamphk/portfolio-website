Some Things I've Built

The Battleship

This project is a recreation of the classic board game Battleship using HTML, CSS, and JavaScript. It allows users to place ships on a game board and take turns guessing the locations of their opponent's ships. The project also demonstrates the use of Webpack for module bundling and organization.

Memory Game

This is a fun and challenging memory game where players click on a set of buttons with emojis but without clicking on the same button twice. With React Hooks, I was able to create a dynamic user interface that updates in real-time as users play the game, such as the Score, Best score and the Level button.

Etch-a-sketch

Introducing my Etch-a-sketch project, a fun and interactive drawing application built with HTML, CSS, and JavaScript. It allows users to create their own digital art by drawing on a virtual canvas with the cursor. With JavaScript, I was able to add functionality that clears the canvas and changes the color of the drawing line.
